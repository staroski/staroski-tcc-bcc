BlueCove Linux D-Bus module.

Introduction

This is additional module for BlueCove to support JSR-82 on Linux BlueZ using D-Bus interfaces.

For JSR-82 API see BlueCove JSR-82 for J2SE implementation

BlueCove bluecove-bluez module expose BlueZ D-Bus API and Unix domain socket API.

  • BlueZ D-Bus API
  • Unix domain socket API
